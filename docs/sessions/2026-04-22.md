## Key decisions

- **No AI-generated imagery anywhere on Basira.** Added to surface
  discipline. The value proposition depends on every pixel being
  real satellite data.
- **Per-GIF tight-fit backdrop over viewport-wide.** The viewport-wide
  approach created a PiP sandwich. Per-GIF gives a seamless rectangular
  visual.
- **CSS masking over re-rendering (Path B over Path A).** Fallback to
  Path A if masking proves fragile in further iterations.
- **Preview PNGs must match GIF dimensions exactly.** object-fit:cover
  on mismatched dimensions shifts the mask off the polygon.
